Research Assistant: Agentic AI Assistant
A powerful, full-stack Agentic AI solution that integrates a Google Chrome Extension with a Spring Boot Microservices backend. This project leverages Spring AI to create an autonomous research assistant capable of processing web content and providing intelligent insights.

Features
Agentic Reasoning: Uses Spring AI to implement autonomous agents that can analyze and summarize complex research data.

Chrome Integration: A custom-built JavaScript extension that captures browser context and interacts with the user in real-time.

Microservices Architecture: Scalable backend built with Spring Boot, ensuring modularity between AI processing and API management.

Seamless Communication: Uses RESTful APIs to bridge the gap between the browser frontend and the Java backend.

Tech Stack
Backend
Java 17+
Spring Boot 3.x
Spring AI: For LLM orchestration and agentic workflows.
Maven: For dependency management.

Frontend (Extension)
JavaScript (ES6+): Logic and API interaction.
Chrome Extension API: Manifest V3 for background scripts and content interaction.
HTML/CSS: For the extension popup UI.

Project Structure
research-assistant/: The Spring Boot microservice handling AI logic and API endpoints.

Research Assistant Extension/: The JavaScript-based Chrome extension files.

pom.xml: Backend dependencies including Spring AI starters.

Getting Started
Prerequisites
JDK 17 or higher
Maven
Google Chrome Browser
OpenAI or Ollama API Key (configured in application.properties)
